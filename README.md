An AI-powered engineering decision support system for designing **Ultra High Performance Concrete (UHPC)** mixes using Machine Learning and Particle Swarm Optimization.

---

## 📖 Overview

The AI-Assisted UHPC Mix Design System is an intelligent engineering application developed to assist researchers and engineers in designing optimized UHPC mixtures.

The system combines:

- 🧠 XGBoost Machine Learning
- ⚙️ Particle Swarm Optimization (PSO)
- 🏗️ Engineering Validation
- 💰 Cost Estimation
- 🌱 CO₂ Assessment
- 📊 Interactive Visualizations

to generate practical, economical, and sustainable UHPC mix designs.

# 🧠 AI Workflow

1. User enters target strength
2. XGBoost predicts achievable strength
3. PSO generates candidate UHPC mixes
4. Engineering validation evaluates practicality
5. Cost and CO₂ are estimated
6. Recommendation engine selects the best mix
7. Results are visualized and exported

---

The system provides:

- Predicted Strength
